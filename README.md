# AI for Handwriting Recognition and Personalized Learning

## Project Description

AI for Handwriting Recognition and Personalized Learning is a web-based application developed to recognize handwritten text, evaluate handwriting quality, and provide personalized learning recommendations. The system helps students improve their handwriting skills while allowing teachers to monitor student performance through a centralized dashboard.

The application is built using Flask for backend development, MySQL for database management, HTML for the user interface, and Tesseract OCR for handwriting text recognition. Students can upload images of handwritten content, and the system extracts the text automatically. Based on the recognized text, the application analyzes handwriting quality and generates personalized recommendations for improvement.

The project includes student registration, handwriting image upload, OCR-based text extraction, quality assessment, recommendation generation, report storage, and teacher report management. All results are stored in a MySQL database for future reference and analysis.

### Key Features

* Student Registration and Management
* Teacher Dashboard
* Handwriting Image Upload
* OCR-based Text Recognition
* Handwriting Quality Analysis
* Personalized Learning Recommendations
* MySQL Database Integration
* Report Generation and Storage
* View Historical Reports

### Technologies Used

* Frontend: HTML, CSS
* Backend: Python Flask
* Database: MySQL
* OCR Engine: Tesseract OCR
* Development Environment: Visual Studio Code

### Project Workflow

Student Uploads Handwritten Image
→ OCR Extracts Text
→ System Analyzes Handwriting Quality
→ Personalized Recommendation Generated
→ Data Stored in MySQL Database
→ Teacher Reviews Reports Through Dashboard
